# Tiny NeRF Experiment

## What I Did
Ran a NeRF model in Google Colab using a synthetic dataset.

## How it Works
- Converts images into rays
- Uses neural network to predict color + density
- Combines results using volume rendering

## Result
Successfully reconstructed a 3D scene and generated a 360° view video.

## Key Learning
- NeRF represents 3D scenes implicitly using neural networks
- Camera poses are critical for learning structure
- Volume rendering converts neural outputs into images

## Output
- Training convergence observed
- Final rendered video generated

## Connection to PreserveMy.World
This technique can be used to digitally preserve heritage sites in 3D without traditional scanning.
